Agent Zero
Distributed constraint reasoning is a framework for describing distributed combinatorial problems of Multi-Agent Systems (MAS). Problems are defined in terms of constraints between variables owned by agents. The agents search cooperatively for a global optimal solution by assigning values to their variables an easy to use, extendible MAS programming framework, execution environment and research tool which specializes on DCR algorithms. AgentZero is designed to be a one stop shop for DCR researchers, a tool that will escort the researcher during every step of his/her algorithmic research.
Ajax (Asynchronous JavaScript and XML) is a group of interrelated web development methods used on the client-side to create interactive web applications. With Ajax, web applications can send data to, and retrieve data from, a server asynchronously (in the background) without interfering with the display and behavior of the existing page. Data is usually retrieved using the XMLHttpRequest object. Despite the name, the use of XML is not needed (JSON is often used instead), and the requests need not be asynchronous.
